2.5.8.5
Protocol-Specific Details
Using RPC
Figure 23: Sending a message to a remote recipient
1. The messaging client submits a message for delivery by using a
RopSubmitMessage
ROP
request (
[MS-OXCROPS]
section 2.2.7.1), per the use case described in section
2.5.7
, except
that the recipient in this case is stored on a different server than the sender.
2. The server determines that the recipient is not stored on the current server and locates the next
hop for message delivery.
3. The server invokes content conversion to convert the message into Internet-transmittable form.
4. The server connects to the destination server and transmits the message via SMTP. For
information regarding SMTP mail submission and Exchange-specific SMTP extensions, see
[RFC2821]
and
[MS-OXSMTP]
