3.4
Example 4: Create an Appointment Request Using Free-Busy Data
3.4.1
Synopsis
This example illustrates how the Microsoft® Exchange Server system can be used to enable an end
user to examine an invitee's free-busy data and send out an appointment request.
Note
This example uses the RPC protocol.
