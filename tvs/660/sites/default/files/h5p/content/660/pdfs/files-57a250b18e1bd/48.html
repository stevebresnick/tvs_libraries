2.5.6
Resolve a Recipient from an Address Book
2.5.6.1
Synopsis
The resolve a recipient from an address book use case describes how a client resolves a recipient
and obtains associated information.
Note
WebDAV requires the client to handle resolving recipient addresses directly and does not have
the corresponding operation.
