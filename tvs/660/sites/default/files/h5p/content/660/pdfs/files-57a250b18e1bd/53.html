2.5.8
Send a Message to a Remote Recipient
2.5.8.1
Synopsis
The send a message to a remote recipient use case describes how a messaging client sends a
message to a recipient that is stored on a remote server. One of the intentions of this use case is to
highlight how content conversion-related protocols are used in message processing. The RPC
protocol is used to illustrate this use case.
