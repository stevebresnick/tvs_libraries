ActiveSync Protocols
Protocols and other technologies listed in this table enable data to be shared and synchronized
between a server and a mobile client device. The ActiveSync protocols also provide a notification
mechanism that allows clients to synchronize updates when changes occur on the server; for
example, when a new e-mail message arrives.
